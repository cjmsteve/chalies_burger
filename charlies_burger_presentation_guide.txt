# 찰리스버거 (Charlie's Burger) 프레젠테이션 제작 가이드

## 디자인 가이드라인
- 색상 팔레트: 찰리스버거 브랜드 색상 활용 (주황색 #FF6B00, 검정색 #333333, 흰색 #FFFFFF)
- 폰트: 제목에는 굵은 고딕체, 본문에는 가독성 좋은 고딕체 사용
- 이미지: 고품질 이미지 사용, 일관된 스타일 유지
- 레이아웃: 깔끔하고 여백이 충분한 디자인, 슬라이드당 정보 과부하 피하기

## 슬라이드별 상세 가이드

### 슬라이드 1: 표지
- 배경: 심플한 단색 또는 햄버거 이미지를 흐리게 배경으로 사용
- 로고: 중앙 상단에 크게 배치 (images/logo.jpeg)
- 제목: "찰리스버거 (Charlie's Burger)" - 크고 굵은 폰트
- 부제목: "프리미엄 햄버거 브랜드" - 제목보다 작은 크기
- 날짜: 하단에 작게 표시

### 슬라이드 2: 목차
- 레이아웃: 왼쪽에 목차 항목, 오른쪽에 관련 작은 이미지 배치
- 각 항목 옆에 작은 아이콘 추가하여 시각적 흥미 유발
- 현재 발표 중인 섹션이 하이라이트되는 인터랙티브 목차 고려

### 슬라이드 3: 브랜드 소개
- 레이아웃: 왼쪽에 텍스트, 오른쪽에 이미지 배치
- 핵심 메시지를 큰 폰트로 강조: "햄버거 덕후가 만든 진짜 수제버거"
- 간결한 3-4개의 핵심 포인트만 포함
- 이미지: 메인 비주얼 (images/main.jpeg) - 고품질, 식욕을 돋우는 이미지

### 슬라이드 4: 브랜드 역사
- 타임라인 형식으로 디자인
- 각 연도별 중요 이벤트를 시각적으로 표현
- 간결한 설명과 함께 작은 이미지 추가
- 애니메이션: 연도별로 순차적으로 나타나는 효과 고려

### 슬라이드 5: 브랜드 철학 및 가치
- 4가지 핵심 가치를 각각 사각형 또는 원형 박스로 표현
- 각 가치마다 관련 아이콘 추가
- 중앙에 "찰리스버거의 약속" 등의 통합 메시지 배치
- 간결하고 기억하기 쉬운 문구 사용

### 슬라이드 6: 메뉴 소개
- 4개 카테고리를 그리드 형태로 배치
- 각 카테고리별 대표 이미지와 간단한 설명
- 클릭 시 해당 카테고리 상세 페이지로 이동하는 인터랙티브 기능 고려
- 배경에 연한 메뉴 관련 패턴 사용 가능

### 슬라이드 7: 인기 메뉴
- 3개 인기 메뉴를 나란히 배치
- 각 메뉴별 고품질 사진, 이름, 가격, 간단한 설명 포함
- "BEST" 또는 "POPULAR" 배지 추가
- 애니메이션: 메뉴가 순차적으로 나타나는 효과

### 슬라이드 8: 매장 안내
- 지도를 중앙에 크게 배치, 각 매장 위치 표시
- 매장별 작은 썸네일 이미지와 주소 정보
- 클릭 시 해당 매장 상세 정보로 확대되는 기능 고려
- 매장 외관 사진 포함

### 슬라이드 9: 운영 시간
- 시각적 타임테이블 형식으로 표현
- 각 매장별 운영시간을 시계 아이콘과 함께 표시
- 브레이크타임은 다른 색상으로 구분
- 특이사항(휴무일 등)은 별도 표시

### 슬라이드 10: 이벤트 및 프로모션
- 3개 이벤트를 카드 형태로 디자인
- 각 이벤트별 이미지, 제목, 기간, 간단한 설명 포함
- "진행중" 배지 추가
- 애니메이션: 카드가 순차적으로 나타나거나 슬라이드되는 효과

### 슬라이드 11: 연락처 정보
- 회사 정보를 깔끔하게 정리
- 로고를 상단에 배치
- 소셜미디어 아이콘 포함
- QR코드 추가 고려 (웹사이트 또는 인스타그램으로 연결)

### 슬라이드 12: 감사합니다
- 심플하고 깔끔한 디자인
- 큰 "감사합니다" 텍스트
- 로고 또는 대표 이미지 배경으로 사용
- 질문을 위한 연락처 정보 포함

## 발표 팁
- 각 슬라이드당 1-2분 내외로 설명
- 핵심 메시지를 중심으로 간결하게 전달
- 청중과 상호작용하는 질문 포함
- 전체 발표 시간: 15-20분 내외로 구성
- 질의응답 시간 5-10분 추가 고려
